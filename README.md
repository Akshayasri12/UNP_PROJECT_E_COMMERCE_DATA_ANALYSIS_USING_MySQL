# UNP_PROJECT_E_COMMERCE_DATA_ANALYSIS_USING_MySQL


I did this project in collaboration with UNP. The project is based on designing an e-commerce database system using SQL. We created a database called smart_cart to manage different activities involved in online shopping such as customers, products, orders, payments, returns, and shipments.
We designed multiple tables with appropriate primary keys and inserted sample data to represent real-world scenarios. We then used various SQL queries like SELECT, JOIN, WHERE, GROUP BY, and ORDER BY to retrieve and analyze the data.
This project helped us understand how data is stored, managed, and analyzed in an e-commerce system, and how SQL can be used to generate meaningful insights.
